# מטבח החסד - ברכת שמואל 🍲

מערכת ניהול קייטרינג קהילתי לשבתות. עברית מלאה, RTL, צבעי מותג מהלוגו.

## מבנה הפרויקט

```
birkat shmuel/
├── supabase/          # מסד נתונים - מיגרציות + seed (כבר רץ בענן ✓)
├── server/            # שרת Node + Express (API)
├── client/            # פרונט React + Vite + Tailwind
└── איפיון_מערכת_מטבח_החסד.md
```

## מה מוכן בגרסה זו

**זרימת הזמנה מלאה מקצה-לקצה:**
- כניסת לקוח לפי טלפון (+ בקשת רישום אם לא קיים)
- אשף הזמנה: בחירת שבת → סעודות ומנות → מאכלים לפי קטגוריה → תוספות → סיכום מחיר חי
- מסך סיכום הזמנה + היסטוריית "ההזמנות שלי"
- ניהול: דשבורד + רשימת הזמנות + אישור / ביטול / עדכון תשלום

## הפעלה - 3 צעדים

### 1️⃣ מפתח סודי (פעם אחת)
פתחי **Supabase → Settings → API**, העתיקי את מפתח **`service_role`** (secret),
והדביקי אותו בקובץ `server/.env`. בנוסף הגדירי `JWT_SECRET` (סוד לחתימת טוקני מנהל):
```
SUPABASE_SERVICE_ROLE_KEY=<הדביקי כאן>
JWT_SECRET=<מחרוזת אקראית ארוכה>
```
לייצור סוד אקראי: `node -e "console.log(require('crypto').randomBytes(48).toString('hex'))"`

### 2️⃣ נתוני דמו (פעם אחת)
```bash
cd server
npm run seed:demo
```
מזין: מחירים, 12 מאכלים, 3 תוספות, 4 שבתות, לקוח דמו לכניסה - **טלפון: 050-123-4567**,
ומנהל דמו - **manager@demo.local / Demo1234!**

להגדרת מנהל אמיתי (או איפוס סיסמה):
```bash
npm run set:admin -- <email> <password> [manager|coordinator|developer] ["שם מלא"]
```

### 3️⃣ הרצה (שני טרמינלים)
```bash
# טרמינל 1 - שרת
cd server && npm run dev
# → http://localhost:3005

# טרמינל 2 - פרונט
cd client && npm run dev
# → http://localhost:5173
```

פתחי **http://localhost:5173** בדפדפן.
